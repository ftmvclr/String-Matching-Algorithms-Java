# String-Matching Algorithm Analysis
This project provides a comparative performance analysis of the Brute-Force, Boyer-Moore, and Horspool string-matching algorithms, implemented in Java.  
It evaluates how each approach handles large-scale text searching by measuring execution time of the total search time within the HTML input.

The repository includes individual implementations for each algorithm alongside a parent class (MatchingAlgorithms.java) designed to measure their efficiency and manage I/O. 
By comparing these methods, the project highlights the practical performance differences between brute force search algorithms and more optimized & sophisticated string searching.

How to run:  
Load the input you would like to test inside input.html.  
The pattern-to-search should go into key.txt.
